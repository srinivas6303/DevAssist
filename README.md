# DevAssist 🚀

DevAssist is a full-stack AI-powered web application built using React and Spring Boot. The project includes secure authentication with Email OTP verification, JWT-based authorization, and AI integration using the Gemini API SDK.

## Features

* Email OTP Registration & Login
* JWT Authentication
* AI Chat Integration using Gemini API
* REST APIs with Spring Boot
* Responsive React Frontend

## Run Project

### Backend

```bash
git clone https://github.com/srinivas6303/DevAssist.git
cd backend
```

Configure:

```properties
google.api.key=YOUR_GEMINI_API_KEY
```

Run Spring Boot application.

### Frontend

```bash
cd frontend
npm install
npm start
```
